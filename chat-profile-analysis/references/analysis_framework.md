# 聊天记录分析框架参考文档

本文档为AI分析提供结构化参考,非死板模板。**请根据实际数据灵活调整报告结构**。

---

## 报告核心结构

```
1. 数据概览（含时间模式）
2. 互动模式分析
3. 语言风格分析
4. 性格特征分析
5. 行为画像
6. 核心理念/价值观
7. 问题发现
8. 改进建议
9. 总结画像
```

---

## 1. 数据概览

### 基础指标
- 总消息数
- 数据来源
- 时间跨度
- 消息密度

### 时间模式分析
| 维度 | 分析要点 |
|-----|---------|
| 活跃时段 | 主要活动的时间段分布 |
| 活跃频率 | 日均/周均消息数 |
| 时间变化 | 随时间推移的使用变化趋势 |

---

## 2. 互动模式分析

### 互动对象
识别用户主要交互对象及其特点

### 互动方式
- 发起方式：指令型/问答型/讨论型
- 回应模式：简短确认/详细追问/直接执行
- 控制程度：主导/协作/跟随

---

## 3. 语言风格分析

### 消息特征
| 特征类型 | 说明 |
|---------|------|
| 简洁指令 | 单词或短语 |
| 详细说明 | 分点说明、背景描述 |
| 情绪表达 | 批评、赞叹、不满 |
| 追问原因 | "为什么"类问题 |

### 沟通倾向
- 直接 vs 委婉
- 简洁 vs 详细
- 理性 vs 感性
- 主动 vs 被动

---

## 4. 性格特征分析

### 大五人格评估

#### 评分标准
| 星级 | 含义 |
|-----|------|
| ⭐⭐⭐⭐⭐ | 极强 - 该特质非常显著 |
| ⭐⭐⭐⭐ | 较强 - 该特质比较明显 |
| ⭐⭐⭐ | 中等 - 该特质有一定表现 |
| ⭐⭐ | 较弱 - 该特质偶尔出现 |
| ⭐ | 极弱/无 - 该特质基本不出现 |

#### 五大维度
1. **外向性** - 社交活跃度、表达欲望
2. **宜人性** - 合作意愿、共情能力
3. **尽责性** - 计划性、注重细节
4. **神经质** - 情绪稳定性、压力反应
5. **开放性** - 接受新事物、创造力

### 核心行为特征（选择适用）
- 深度思考型
- 风险意识强
- 追求完美/高要求
- 技术导向
- 迭代改进型
- 结果导向型
- 独立自主型

### 典型行为模式
**模式格式**：
- 触发条件
- 典型表现（引用原话）
- 行为解读
- 出现频率

## 重要原则

1. **数据驱动**：所有结论必须基于实际聊天内容
2. **引用原话**：关键判断需引用原文支撑
3. **动态调整**：根据数据特点调整报告结构
4. **具体可行**：建议要具体、可操作
5. **客观中立**：避免主观臆断
